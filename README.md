## عن المشروع

متجر **AL MNEE** هو موقع إلكتروني متكامل لعرض وبيع الاكسسوارات النسائية الفاخرة. تم تصميمه بعناية ليقدم تجربة تسوق أنيقة ومميزة.

## المميزات

- 🎨 تصميم عصري وأنيق مع دعم RTL للعربية
- 📱 تصميم متجاوب يعمل على جميع الأجهزة
- 🛒 سلة تسوق تفاعلية
- 🔍 تصفية المنتجات حسب الفئات
- ✨ رسوم متحركة وتأثيرات بصرية جذابة
- 📧 نموذج اشتراك في النشرة البريدية
- 📱 أيقونات التواصل الاجتماعي

## التقنيات المستخدمة

- HTML5
- CSS3 (مع متغيرات CSS)
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Cairo & Playfair Display)

## التنصيب

1. استنساخ المستودع:
```bash
git clone https://github.com/YOUR_USERNAME/al-mnee-website.git
```

2. افتح الملف `index.html` في المتصفح

## الأقسام الرئيسية

- **الرئيسية**: عرض ترحيبي مع زر للتصفح
- **الفئات**: تصنيفات المنتجات
- **المنتجات**: عرض المنتجات مع إمكانية الإضافة للسلة
- **من نحن**: إحصائيات ودراسات حالة
- **النشرة البريدية**: نموذج اشتراك
- **التذييل**: روابط سريعة ومعلومات التواصل

## التخصيص

يمكنك تخصيص الألوان من خلال متغيرات CSS في بداية ملف `index.html`:

```css
    --elegant-black: #1a1a1a;

}
```
